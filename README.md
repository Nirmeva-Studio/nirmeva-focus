# Nirmeva Focus

A macOS menu bar timer engineered for deep work.

![Nirmeva Focus Screenshot](https://product.nirmeva.com/images/product/nirmeva-focus/v1.2.0/screenshot-timer-active.png)

---

## Features

- **Menu bar only** — no Dock icon, no distractions
- **Pomodoro-style sessions** with automatic break overlays
- **Health reminders** — 20-20-20 eye strain, posture check, stretch break, water break
- **Offline analytics** — weekly and monthly focus history
- **100% private** — no account, no internet required
- **Free**

---

## Download

**Homebrew (Recommended)**
```bash
brew install --cask nirmeva-studio/tap/nirmeva-focus
```

**Manual DMG**  
Download the latest release from [nirmeva-products/releases](https://github.com/Nirmeva-Studio/nirmeva-products/releases)

After manual install, run this once in Terminal:
```bash
xattr -cr /Applications/Nirmeva\ Focus.app
```

---

## Requirements

- macOS 10.12 or later
- Apple Silicon (M1/M2/M3/M4)
